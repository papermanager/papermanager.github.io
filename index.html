Paper Manager

Academic publication manager for Joomla! - Component & plugin.

GitHubDownloadGet Started
⭐️ If you like Paper Manager, give it a star on GitHub ⭐️
Paper Manager is developed for Joomla 4.x and any future updates will be checked for compatibility only with 4.x.
If you need to install Paper Manager on Joomla 3.x, use version 0.0.1-joomla3.x.

Manage

Manage academic publications, authors and categories (e.g. conference publications, journal publications).

Assign

Assign the publications to authors and categories.

Query

Query the publications and generate custom filtered and sorted publication lists.
